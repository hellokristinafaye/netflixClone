Overview:

- Netflix Clone that links to a web API to provide clips from the movies.

Technologies Used:

- React
- TheMovieDB.org API
- Vite
- JavaScript
- CSS

Features:

- Smooth Scrolling
- Each movie has its own dynamically and quickly produced details page, featuring a clip from TheMovieDB API
